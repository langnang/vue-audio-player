# vue-audio-player

> Vue 音频播放器组件

## 安装

```bash
npm i -S @langnang/vue-audio-player
```

## 按需引入
```javascript
import { AudioPlayer } from '@langnang/vue-audio-player'
import '@langnang/vue-audio-player/lib/vue-audio-player.css'

components: {
  AudioPlayer
}
```

## 全局引入
```javascript
import AudioPlayer from '@langnang/vue-audio-player'
import '@langnang/vue-audio-player/lib/vue-audio-player.css'

Vue.use(AudioPlayer)
```
